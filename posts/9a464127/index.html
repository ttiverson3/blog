Looking back on 2022

Created2022-12-26|Updated2023-01-05
|Post View:

2022 上半年

年初時公司預計之後會慢慢停更某些舊產品，於是我架了一台 R Studio + R Shiny 的 server，蒐集 update server 上每日的產品資料做統計分析與視覺化，以供上層做決策。

約莫三月的時候，為了因應 PM 們作業上的需求，我基於目前公司內部使用的 Mantis Bug Tracker，實做出另一套符合 PM 需求的客製化 issue 管理系統，結合原有的 Mantis Bug Tracker 資料庫，讓 PM 能夠自行新增專案並對 issue 進行分類、修改、註記、匯出等…，並能依照 issue 狀態繪製進度條，讓 PM 隨時掌握工作進度並節省作業成本，也是我初次使用 Vue 來做 UI 並搭配 axios 來處理 AJAX，並得到不錯的反饋。

年中時收到 AWS 的通知說 EC2-Classic 將停止服務，所以我必須在期限內完成 VM 轉換，我負責的部分總共四台，由於這些都是公司客戶會用到的服務，加上有時間限制，再看到每台 VM 都是 Ubuntu 12 + PHP 5，可以肯定這些服務以前從來沒有更新過，所以能在期限內完成系統升級，並無縫接軌的保持正常運作，算是今年最有感覺的一項工作。

2022 下半年

因為有了比較多的個人時間，我開始思考除了工作外我還能做什麼，我喜歡做什麼？我想到大學時曾經參加過半學期的熱舞社，我一直覺得熱衷於跳舞的人很有吸引力，於是我從 7 月開始，每週都會去 Dance Studio 跳舞，另外也跟著朋友們一起健身運動。

12/18 我跟朋友搶到了 8 張桃園雲豹的主場門票，準備一睹魔獸 Dwight Howard 的比賽，可惜的是當天他並沒有上場，這是我第一次到現場看比賽，不得不說真的有點可惜，但現場的臨場感很明顯跟在螢幕前觀看是完全不一樣的。

公司使用的系統，大部分都是 PHP CodeIgniter 框架，但目前比較主流的是 Laravel 框架，我很幸運的是進到公司，不只是在維護各種 legacy code，我從入職到現在已經獨立開發了兩個公司內部系統，而目前又有一個新系統在規劃，我開始嘗試 Vue + Laravel + Docker 的組合，很高興能夠學習新技術並實際做出貢獻。

2022 總回顧

今年對我來說是一個新開始，花了不少時間在與自己對話，不斷地摸索與嘗試，試著找出一個自己喜歡的生活模式，而這個過程還在持續中。我以前很少紀錄生活中發生的大小事，期許 2023 一樣能保持「記錄」這個習慣，不管是透過文字、聲音、影像、圖片等…。
