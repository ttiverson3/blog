GitHub Pages 套用自定網域及 SSL

Created2022-09-28|Updated2022-09-29|DevelopmentHexo
|Post View:

原本架在 GitHub Pages 上的舊網域: https://ttiverson3.github.io/blog
自定的新網域: https://blog.garyhuang.tw
現在只要點選舊網域都會自動導向到新網域。

購買網域

找個網域名稱註冊商購買網域，我是使用 Godaddy，註冊的網域為 garyhuang.tw

使用 Cloudflare 的免費 SSL

Cloudflare 註冊會員，點選 add site 加入購買的網域。

點選最底下的 Free 方案

複製以下兩個 Cloudflare’s nameservers

至 Godaddy DNS 管理介面變更名稱伺服器

套用以下設定: HTTPS Rewrites 會將 HTTP 改為 HTTPS

到管理面板點選 SSL/TLS 確認 SSL 為 Full

到管理面板點選 DNS 新增一個 CNAME record 指到 GitHub Pages

Hexo GitHub Pages 設定

ref: Configuring a subdomain
到個人 Hexo blog repository 的 setting，點選側邊攔的 Pages 在 Custom domain 加上自定的網域

最後在 Hexo 專案底下的 source 目錄新增一個 CNAME 文件，內容為自定的網域。
