Laravel9 使用 Vite 安裝 Vue3

Created2022-12-22|Updated2023-01-05|DevelopmentLaravelVue.js
|Post View:

前置作業

請參考：https://blog.garyhuang.tw/posts/7fbb1596/
運行環境：Ubuntu 22.04、Laravel 9
目錄結構：

安裝 Vite

ref: https://laravel.com/docs/9.x/vite#installation

  1. 安裝 Node.js (16+)、NPM
    ref: https://github.com/nodesource/distributions/blob/master/README.md#debinstall
    注意要安裝 16 以上的版本才行，這裡安裝 v16.x 版 檢查版本
  2. 安裝 Vite、Laravel plugin
    在 Laravel 目錄底下有一個預設的 package.json 檔，包含了 Vite 及 plugin 需要的 packages，使用 NPM 安裝
  3. 運行 Vite
    輸入網址後可以看到 Vite 已經安裝完成

安裝 Vue.js

ref: https://laravel.com/docs/9.x/vite#vue

  1. 安裝 @vitejs/plugin-vue
  2. 在專案目錄底下的 vite.config.js 引入 vue

Vue 與 Laravel 整合

ref: https://techvblogs.com/blog/how-to-install-vue-3-in-laravel-9-with-vite

  1. 運行 dev server

  2. 運行 Laradock

  3. 新增 route
    /blog/routes/web.php

  4. 新增 view
    /blog/resources/views/app.blade.php

  5. 新增 App.vue
    /blog/resources/js/App.vue

  6. 將 Vue component mount 到畫面上
    /blog/resources/js/app.js

    最後連到網頁確認畫面成功渲染
